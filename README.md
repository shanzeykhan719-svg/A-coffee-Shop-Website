# ☕ The Debug Cafe – Coffee Shop Website

A modern and responsive coffee shop website built using HTML, CSS, and JavaScript.  
This project showcases a clean UI design, menu display, and smooth user experience.

## 📌 Features

- Responsive design (Mobile, Tablet, Desktop)
- Attractive hero section
- Coffee menu with pricing
- About section
- Contact form
- Smooth scrolling navigation
- Clean and modern UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
